# 관내여비 콕검 v0.2.8

에듀파인에서 출력한 관내여비 내역 또는 대상목록(`.xls`, `.xlsx`)을 브라우저에서 읽고, 근무지와 출장지의 **보행 왕복거리와 2km 여부를 한눈에 확인**하는 업무 보조 웹앱입니다.

## v0.2.8 핵심 변경

- 메인 문구를 `관내출장 왕복 2km, 한눈에 확인`으로 변경
- 개인정보 보호 안내를 두 줄의 강조 안내 바로 개선
- 푸터에 `제작 senvip`과 GitHub 링크 추가
- 파일 업로드 영역에 에듀파인 엑셀 저장 경로 안내 추가
  - 학교회계 > 지출관리 > 지출지원 > 출장여비 > 출장여비(관내) > [출력] > 엑셀 저장
  - 교특회계: 재정사업관리 > 사업재정관리 > 재무관리 > 관내여비관리 > 엑셀 저장
- 원본 출장지와 지도 검색어를 분리
- `1층`, `3층 회의실`, `대강당`, `시청각실` 등 건물 내부 위치를 지도 검색 시 자동 제외
- `본관`, `별관` 같은 건물 구분은 보존
- 도로명주소의 숫자와 번지는 그대로 유지
- 화면과 결과 엑셀에 원본 출장지와 정리된 지도 검색어를 함께 보존
- 장소 자동 확정 시 정리된 지도 검색어를 기준으로 일치 여부 판단

- v0.3의 출장자·날짜별 지급 검토 작업대 제거
- 이동수단 선택, 검토 메모, 완료·보류 처리 제거
- v0.2 일괄 거리검사 흐름 복구
- 출장별 결과표를 6열로 단순화
  - 출장일
  - 출장자
  - 출장지·목적
  - 왕복거리
  - 2km 판정
  - 확인사항
- 출장지와 출장목적을 한 셀의 두 줄로 통합
- PC 결과표의 가로 스크롤 제거
- 모바일에서는 결과 행을 카드형으로 자동 전환
- 왕복 2km 이내는 `이동수단 확인 필요`로만 안내
- 위치 미확인, 거리 계산 실패, 경계 구간을 자동 안내
- 상단 요약 숫자를 눌러 바로 필터 가능

> 이 앱은 지급액을 확정하지 않습니다. 급여담당자가 왕복 2km 이내 출장과 위치·거리 확인이 필요한 출장만 빠르게 찾도록 돕습니다.

## 주요 기능

- 에듀파인 `.xls`·`.xlsx` 파일 업로드
- 인쇄용 페이지 번호, 기관명, 합계 행 자동 제외
- 동일 출장지 자동 묶기
- 모호한 장소 자동 확정 방지
- 근무지와 출장지 위치를 브라우저에 저장
- TMAP 보행자 경로 API로 갈 때·올 때 거리 각각 계산
- 왕복 2,000m 기준 판정
- 1,900~2,100m 경계 구간 별도 표시
- 출장지 일괄검색 및 일괄거리 계산
- 미완료 항목만 재검사
- 결과 엑셀 내려받기

## 개인정보 처리

- 업로드한 엑셀 파일은 브라우저에서만 처리합니다.
- 지도 API에는 장소 검색어 또는 출발·도착 좌표만 전달합니다.
- 출장자명, 출장목적, 여비금액, 지급금액은 지도 API로 보내지 않습니다.
- 근무지와 사용자가 확정한 출장지 위치는 해당 브라우저의 `localStorage`에 저장됩니다.

## 실행 방법

```bash
npm install
npm run dev
```

지도 API까지 로컬에서 확인하려면 Vercel CLI를 사용합니다.

```bash
npx vercel dev
```

프로덕션 빌드:

```bash
npm run build
```

## TMAP API 설정

1. SK Open API에서 앱을 생성합니다.
2. 해당 앱에 TMAP 상품의 무료 요금제를 연결합니다.
3. 발급받은 App Key를 Vercel 환경변수에 등록합니다.

```text
TMAP_APP_KEY=발급받은_앱키
```

4. Vercel에서 다시 배포합니다.

## Vercel 배포

1. 프로젝트 파일을 GitHub 저장소에 올립니다.
2. Vercel에서 저장소를 가져옵니다.
3. `TMAP_APP_KEY`를 `Production and Preview` 환경에 등록합니다.
4. Deploy 또는 Redeploy를 실행합니다.

## 테스트

```bash
npm test
npm run build
```

포함된 비식별 테스트 파일은 `.xls`와 `.xlsx` 모두 동일하게 다음 결과로 파싱됩니다.

- 관내출장 4건
- 출장자 4명
- 출장지 3곳

기존 관내여비 내역 파일은 `.xls`와 `.xlsx` 모두 **관내출장 30건 · 출장자 16명 · 출장지 17곳**으로 동일하게 확인했습니다. 추가로 제공된 `관내여비대상목록.xlsx`도 **관내출장 54건 · 출장자 15명 · 출장지 28곳**으로 정상 인식했습니다. 실제 파일은 개인정보 보호를 위해 ZIP에 포함하지 않습니다.
